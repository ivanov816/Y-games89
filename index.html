🔒 Политика конфиденциальности

Версия: 2.0 | Дата: 01.05.2026

1. Какие данные мы собираем

Y Games НЕ собирает, НЕ хранит и НЕ передаёт никакие персональные данные. Платформа работает полностью офлайн, весь код выполняется локально на устройстве Пользователя. Никакая информация не отправляется на сервера.

2. Локальное хранение

Браузер может сохранять файл .html локально для последующего использования. Это стандартная функция браузеров, не контролируемая платформой. Пользователь может удалить файл в любой момент.

3. Сторонние библиотеки

Платформа использует CDN для загрузки Three.js и Cannon.js (физический движок). При загрузке этих библиотек ваш браузер может передавать стандартные технические данные (IP, тип браузера) на сервера CDN. Y Games не контролирует эти сервера. Рекомендуем ознакомиться с политикой конфиденциальности Cloudflare CDN.

4. Пользовательский код

При вставке собственного HTML-кода в поле «Своя игра» весь код выполняется локально и не передаётся третьим лицам. Вы несёте полную ответственность за содержимое вставляемых скриптов.

5. Права пользователя

  • Полное удаление платформы — просто удалите HTML-файл.
  • Очистка кэша браузера удалит все временные данные.
  • Никаких запросов на обработку данных не требуется, т.к. данные не обрабатываются.

6. Согласие

Используя платформу, вы подтверждаете, что ознакомлены с данной Политикой конфиденциальности и принимаете её условия.
